Load Test Report: Helios Login Endpoint

Author: Mei Tanaka (Performance Engineer)
Date: 2026-03-18
Status: Final

1. Scenario

login load test: a sustained load test of the Helios login endpoint, ramping concurrent
clients until throughput plateaued. The scenario drives the POST /login path end to end.

2. Target

This load test validates REQ-4: the login endpoint sustains 10,000 requests/second at
p99 < 200 ms. The target value is 10,000 requests/second at p99 < 200 ms.

3. Observed Results

The login load test reached 8,000 req/s against the 10,000 req/s target. Observed
throughput was 8,000 requests/second. Observed p99 latency was 250 ms against the
< 200 ms target.

4. Verdict

Result: FAILED. The login load test missed the target. Observed throughput of 8,000
requests/second is below the 10,000 requests/second target, and observed p99 latency of
250 ms exceeds the 200 ms target. The login endpoint did NOT meet the PRD target.

5. Follow-up

Tracking a performance work item against the auth-service to close the gap to the
10,000 requests/second target before launch.
